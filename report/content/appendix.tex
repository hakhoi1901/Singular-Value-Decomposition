\subsection{Hướng dẫn cài đặt và chạy thử}

Để tái tạo lại toàn bộ kết quả thực nghiệm và kiểm tra hoạt động của từng phân hệ thuật toán, hệ thống yêu cầu môi trường \textbf{Python 3.10 trở lên}. Quá trình triển khai được thực hiện tuần tự qua các bước sau.

\begin{enumerate}

    % ---
    \item \textbf{Clone dự án từ kho lưu trữ GitHub}

    Sử dụng \texttt{git} để tải toàn bộ mã nguồn về máy cục bộ:
\begin{tcolorbox}[mycodebox]
\begin{verbatim}
git clone https://github.com/hakhoi1901/Singular-Value-Decomposition.git
cd "Singular Value Decomposition"
\end{verbatim}
\end{tcolorbox}

    % ---
    \item \textbf{Tạo và kích hoạt môi trường ảo}

\begin{tcolorbox}[mycodebox]
\begin{verbatim}
python -m venv venv

:: Windows
venv\Scripts\activate

# Linux / macOS
source venv/bin/activate
\end{verbatim}
\end{tcolorbox}

    % ---
    \item \textbf{Cài đặt các thư viện phụ thuộc}

    Toàn bộ thư viện yêu cầu (bao gồm \texttt{numpy}, \texttt{manim}, \texttt{matplotlib}, \texttt{jupyter}) đã được
    liệt kê trong tệp \texttt{requirements.txt}. Chạy lệnh sau để cài đặt một lần:
\begin{tcolorbox}[mycodebox]
\begin{verbatim}
pip install -r requirements.txt
\end{verbatim}
\end{tcolorbox}

    \textbf{Lưu ý quan trọng:} \texttt{NumPy} trong dự án này \emph{chỉ} được dùng để xác minh kết quả kiểm thử
    và cung cấp dữ liệu đầu vào cho Manim. Toàn bộ thuật toán lõi (khử Gauss, tính trị riêng, phân rã SVD)
    được cài đặt bằng Python thuần, không phụ thuộc vào bất kỳ hàm phân rã nào của NumPy/SciPy.

    % ---
    \item \textbf{Kiểm thử từng module đơn lẻ (Unit Tests)}

    Mỗi module trong từng phần đều có hàm kiểm thử riêng. Chạy trực tiếp file Python tương ứng
    để xem kết quả PASSED/FAILED cho từng ca kiểm thử:

    \textit{Phần 1 -- Khử Gauss và các ứng dụng:}
\begin{tcolorbox}[mycodebox]
\begin{verbatim}
python part1/gaussian.py
python part1/back_substitution.py
python part1/determinant.py
python part1/inverse.py
python part1/rank_basis.py
python part1/verify_solution.py
\end{verbatim}
\end{tcolorbox}

    \textit{Phần 1 -- Chạy toàn bộ test:}

\begin{tcolorbox}[mycodebox]
\begin{verbatim}
python part1/run_all_tests.py
\end{verbatim}
\end{tcolorbox}

    \textit{Phần 2 -- Chéo hóa và Phân rã SVD:}
\begin{tcolorbox}[mycodebox]
\begin{verbatim}
python part2/decomposition.py
python part2/diagonalization.py
\end{verbatim}
\end{tcolorbox}

    % ---
    \item \textbf{Chạy toàn bộ hệ thống kiểm thử tự động (Part 1)}

    File \texttt{run\_all\_tests.py} trong thư mục \texttt{part1/} tự động thu gom và chạy
    tuần tự toàn bộ 6 bộ kiểm thử của Phần 1, bao gồm:
    \texttt{gaussian\_eliminate}, \texttt{back\_substitution}, \texttt{determinant},
    \texttt{rank\_and\_basis}, \texttt{inverse} và \texttt{verify\_solution}:
\begin{tcolorbox}[mycodebox]
\begin{verbatim}
cd part1
python run_all_tests.py
\end{verbatim}
\end{tcolorbox}

    % ---
    \item \textbf{Khởi chạy kịch bản Benchmark và xuất dữ liệu JSON (Part 3)}

    Script \texttt{time\_benchmark.py} thực hiện đo lường hiệu năng thời gian thực thi
    của ba bộ giải (Gauss LU, SVD, Gauss-Seidel) trên nhiều kích thước ma trận,
    lặp lại 5 lần (\texttt{N\_RUNS = 5}) để loại bỏ nhiễu hệ điều hành, rồi xuất kết quả
    ra file \texttt{benchmark\_results.json} phục vụ Jupyter Notebook đọc và vẽ biểu đồ:
\begin{tcolorbox}[mycodebox]
\begin{verbatim}
python part3/time_benchmark.py
\end{verbatim}
\end{tcolorbox}

    Để xem phân tích trực quan hóa đầy đủ (log-log chart, bảng sai số tương đối), mở Jupyter Notebook:
\begin{tcolorbox}[mycodebox]
\begin{verbatim}
jupyter notebook
# Mo file: part3/analysis.ipynb
\end{verbatim}
\end{tcolorbox}

    % ---
    \item \textbf{Render video trực quan hóa Manim (tùy chọn)}

    Scene Manim trong \texttt{part2/manim\_scene.py} trực quan hóa ý nghĩa hình học của phép
    phân rã $A = U\Sigma V^T$ theo tiếp cận ``Rotate -- Scale -- Rotate''. Yêu cầu đã cài đặt
    \texttt{manim} và \texttt{FFmpeg}. Các lệnh render theo chất lượng:
\begin{tcolorbox}[mycodebox]
\begin{verbatim}
# Xem truoc nhanh (480p)
manim -pql part2/manim_scene.py Scene1_AnatomyOfChaos

# Chat luong cao (1080p60)
manim -pqh part2/manim_scene.py Scene1_AnatomyOfChaos

# Xuat GIF
manim -pql --format=gif part2/manim_scene.py Scene1_AnatomyOfChaos
\end{verbatim}
\end{tcolorbox}

    Output video được lưu tự động tại \texttt{part2/media/videos/}.

    % ---
    \item \textbf{Biên dịch báo cáo LaTeX}

    Để tái tạo lại file PDF từ mã nguồn \LaTeX{}, chạy \texttt{pdflatex} hai lần
    (lần một để sinh nội dung, lần hai để cập nhật tham chiếu chéo và mục lục):
\begin{tcolorbox}[mycodebox]
\begin{verbatim}
cd report
pdflatex report.tex
pdflatex report.tex
\end{verbatim}
\end{tcolorbox}

\end{enumerate}

\subsection{Chi tiết Log kiểm thử thuật toán}

Phần dưới đây cung cấp log trích xuất nguyên bản từ Terminal khi chạy bộ kiểm thử
đầy đủ của Phần 1. Các ca kiểm thử bao gồm các tình huống hóc búa nhất về mặt số học:
ma trận cần \textit{Partial Pivoting}, hiện tượng \textit{swamping} (chênh lệch hệ số
khổng lồ), ma trận Hilbert (số điều kiện cực lớn) và ma trận suy biến (thiếu hạng).
Chuỗi log này là minh chứng trực tiếp cho năng lực ổn định số học của các thuật toán
tự cài đặt trong dự án.
